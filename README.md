# AI Universe | Exploring the Neural Frontier 🌌

An ultra-modern, minimalist, and responsive single-page landing page that showcases leading Large Language Models (LLMs) and categorizes different types of Artificial Intelligence. Built with a premium, futuristic dark-theme design leveraging clean semantic HTML5 and vanilla CSS3 architecture.

## 🚀 Live Demo

https://astonishing-pegasus-b83c10.netlify.app/

---

## ✨ Features

- **Futuristic Tech Theme:** A premium dark UI built with custom CSS variables, sharp borders, and subtle glows curated for an AI-focused platform.
- **The Intelligence Layer:** A sleek comparison grid highlighting top industry models (ChatGPT, Claude, Gemini, Llama) with custom interactive hover experiences.
- **AI Classification Matrix:** An organized layout defining the core evolutionary branches of AI from Narrow AI up to Artificial General Intelligence (AGI).
- **Clean Architecture:** Production-ready code structure completely decoupled into independent modules (`style.css`, `responsev.css`, `animation.css`).
- **Fully Responsive & Fluid:** Tailored multi-breakpoint media queries ensuring pixel-perfect layout adaptability across Ultra-wide monitors, Tablets, and Smartphones.
- **Native UI Controls:** Built-in mobile sidebar navigation powered exclusively by CSS (`#menu-toggle` checkbox hack) without relying on heavy external JavaScript bundles.

---

## 🛠️ Tech Stack & Resources

- **Structure:** Semantic HTML5 elements (`<header>`, `<main>`, `<section>`, `<article>`) optimizing accessibility (ARIA attributes) and SEO.
- **Styling:** Modern CSS3 featuring Flexbox, Grid layouts, custom CSS custom properties (`:root`), and advanced transition curves.
- **Typography & Icons:** Integrated Google Fonts (`Segoe UI` fallback) and Google Material Symbols (Outlined variant).
